# Security Policy

## Supported Versions

This project is currently maintained on the `main` branch.

## Reporting a Vulnerability

Please report vulnerabilities privately.

- Preferred: open a private security advisory on GitHub for this repository
- Fallback: contact the maintainer directly and include reproduction details

Please include:

- affected version/commit
- clear reproduction steps
- impact assessment
- suggested fix (if available)

You can expect an acknowledgement within 72 hours for valid reports.

## Disclosure

Please avoid public disclosure until a fix is available and users can patch safely.
